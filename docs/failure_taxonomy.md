# Failure Taxonomy

## 1. Instruction hijacking
The agent follows malicious context over the system objective.

## 2. Unsafe execution
The agent attempts or endorses a risky action it should avoid.

## 3. Objective drift
The agent gradually loses the original task boundary.

## 4. Over-refusal
A safeguard blocks benign behavior too often.

## 5. Missed intervention
A safeguard fails to stop clearly unsafe behavior.
